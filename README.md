# vaccination_app


## This app is designed to help authorities to schedule vaccination appointments with Users.

---

following steps are necessary to get the
application up and running.



* Ruby version => 2.6.6

* Database - Postgres

* Configuration

To run this app locally: 
```
bundle install
```
```
 rails db:setup
```
```
rails s
```
